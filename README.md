# App Rifa Flutter

## Descripción

Aplicación desarrollada en Flutter que permite visualizar una lista de números del 01 al 20 para una rifa digital.
El usuario puede seleccionar un número, ver su estado y reservarlo mediante una confirmación.

## Funcionalidades

* Visualización de números del 01 al 20
* Estado de cada número: Disponible / Reservado
* Navegación entre pantallas
* Confirmación de reserva con diálogo

## Pantallas implementadas

* Home (pantalla principal)
* Lista de números
* Detalle del número
* Confirmación de reserva
